# 🚗 Dedge Service — The Muscle Garage

A modern muscle car themed web application built using **Node.js, Express.js, Handlebars, MongoDB Atlas, and Bootstrap**.
This project showcases a stylish automotive service website with dynamic content fetched directly from MongoDB.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/techbydeepak/dedge_service.git
```

---

## 2️⃣ Move Into Project Folder

```bash
cd dedge_service
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Create `.env` File

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5556
NODE_ENV=development
```

---

## 5️⃣ Start Development Server

```bash
npm start
```

---

# 🚀 Deployment

This project is deployed using **Render**.

## Deployment Steps

1. Push project to GitHub
2. Create a new Web Service on Render
3. Connect GitHub repository
4. Add Environment Variables
5. Deploy 🚀
